# Матрица полей — черновик на ревизию

> **Статус: `[ЧЕРНОВИК]`** — собрано Claude ревизией моделей + движка + вьюх
> (волна консистентности, 2026-07-05). Приложение к [UI_GUIDE.md](UI_GUIDE.md) §6.
> **Задача Ивана:** пройти по таблицам, в колонке **`?`** пометить спорные ячейки
> и поправить «Предлагается», где не согласен. После вычитки — фолдим в UI_GUIDE §6
> и реализуем расхождения (где **Сейчас ≠ Предлагается**).

Принцип §6: *редактируемо всё, что хранится в документе, — включая автора, даты и
структурные якоря*. Исключения ровно три: суррогатный **id**, **вычисленные
проекции линзы** и поля **зафиксированных** документов (до снятия фиксации).

## Легенда классов

| Класс | Значение |
|-------|----------|
| **✎** | редактируемо под **замком формы** §5 (у документа гаснет при фиксации) |
| **⚓** | структурный **якорь**: редактируемо под замком, но движок валидирует и **может отказать** при зависимых (§6) |
| **＋** | **строка-ссылка**: задаётся при добавлении (в призрачной строке); в готовой строке не меняется — **только удалить + добавить** (принято 2026-07-05) |
| **○** | **read-only** — вычисленная проекция линзы (правится в документе-факте), не правится нигде |
| **–** | **скрыто / нет на форме** (id, служебное, авто-выводимое) |
| **N/A** | поля на этой форме нет |

- **Сейчас** = фактическое поведение в коде (движок + PATCH-вьюхи).
- **Предлагается** = целевое по принципу §6.
- Строки, где **Сейчас ≠ Предлагается**, — очередь реализации; они же чаще всего спорные.
- «фикс.» в скобках у документа = какое состояние делает форму read-only
  (approved / posted / closed / status≠draft).
